The format of the UniqueId type for rbxl and rbxlx files.

```go
// Binary format.
type UniqueId struct {
	// All big-endian.
	Sequence  uint32
	Timestamp uint32
	Random    int64  // Zigzag encoded
}
// Interleaved


// XML hash
//
// [random        ][time  ][seq   ]
// 0123456789abcdef0123456789abcdef
//
// Not zigzag encoded!
```
